# Parse-GPB

Парсер данных из Федерального информационного фонда по обеспечению единства измерений (ФГИС Гост Р) для метрологических средств.

## Описание

Этот проект предназначен для автоматического сбора и обработки данных о средствах измерений из базы данных ФГИС Гост Р. Программа считывает идентификаторы (ID) из ODS-файла, запрашивает подробную информацию с сайта fgis.gost.ru и сохраняет результаты обратно в файл.

## Требования

- Операционная система: Windows, macOS или Linux
- Python 3.8 или выше
- Стабильное интернет-соединение

## Установка

### Шаг 1: Установка Python

1. Перейдите на официальный сайт Python: https://www.python.org/downloads/
2. Скачайте и установите последнюю версию Python 3.x
3. Во время установки обязательно отметьте опцию "Add Python to PATH"
4. Проверьте установку, открыв командную строку и введя:
   ```
   python --version
   ```
   Должно отобразиться что-то вроде "Python 3.11.0"

### Шаг 2: Скачивание проекта

1. Скачайте ZIP-архив проекта с GitHub или клонируйте репозиторий:
   ```
   git clone https://github.com/your-username/parse-gpb.git
   ```
   (Замените URL на актуальный адрес репозитория)

2. Распакуйте архив в удобную папку, например `C:\parse-gpb`

### Шаг 3: Установка зависимостей

1. Откройте командную строку (Win+R → cmd) или PowerShell
2. Перейдите в папку проекта:
   ```
   cd C:\parse-gpb
   ```
3. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

## Подготовка данных

1. Поместите ваш ODS-файл с данными в папку проекта
2. Убедитесь, что в файле есть колонка с ID (обычно 15-я колонка, индекс 14)
3. Файл должен иметь расширение `.ods`

## Запуск программы

### Базовый запуск

В командной строке, находясь в папке проекта:
```
python main.py
```

### Запуск с указанием начальной строки

Если обработка прервалась, можно продолжить с определенной строки:
```
python main.py 867
```
(где 867 - номер строки, с которой начать обработку)

## Как это работает

1. Программа считывает все ID из указанного ODS-файла
2. Для каждого ID выполняет запрос к сайту fgis.gost.ru
3. Парсит полученную HTML-страницу и извлекает информацию о средстве измерений
4. Сохраняет данные в соответствующие колонки ODS-файла
5. Каждые 50 запросов перезапускает сессию для избежания блокировок

## Структура проекта

- `main.py` - основной скрипт запуска
- `parser.py` - класс для парсинга веб-страниц
- `ExcelWorker.py` - класс для работы с ODS-файлами
- `Card.py` - типизация данных о средствах измерений
- `requirements.txt` - список зависимостей Python

## Возможные проблемы и решения

### Проблема: "python не является внутренней или внешней командой"
**Решение:** Python не добавлен в PATH. Переустановите Python с отметкой "Add Python to PATH" или добавьте путь вручную.

### Проблема: Ошибка импорта модулей
**Решение:** Убедитесь, что все зависимости установлены командой `pip install -r requirements.txt`

### Проблема: Файл ODS не найден
**Решение:** Проверьте, что файл находится в папке проекта и имеет правильное имя.

### Проблема: Сайт блокирует запросы
**Решение:** Программа имеет встроенные задержки и перезапуски сессии. Если проблема persists, попробуйте запустить позже или с другого IP.
